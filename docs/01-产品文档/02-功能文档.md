# 家居布局画板功能文档

## 1. 文档说明

- 文档状态：当前功能说明
- 适用版本：`zhijian-hub@0.1.0`
- 最后核对：2026-07-18
- 维护范围：当前页面功能、操作方式和用户可见行为
- 关联代码：`src/components/features/room-canvas/`、`src/components/ui/`

本文档面向产品、测试和开发人员，说明当前页面已经提供的功能、操作方式和行为规则。

## 2. 页面组成

当前项目只有一个家居布局画板页面，主要由以下区域组成：

| 区域       | 位置         | 作用                                    |
| ---------- | ------------ | --------------------------------------- |
| 品牌标识   | 左上角       | 展示“知简”品牌                          |
| 顶部工具栏 | 顶部居中     | 切换工具、添加房间、撤销/重做、清空画板 |
| 画板功能区 | 右上角       | 搜索当前实体、查看房间和当前位置详情    |
| 画布       | 页面主体     | 绘制和编辑房间、家具及房间级储物设备    |
| 缩放控件   | 右下角       | 缩小、显示比例、放大                    |
| 右键菜单   | 指针位置附近 | 执行画板或空间实体操作                  |
| 重命名弹窗 | 页面中央     | 修改房间、家具或储物设备名称            |

## 3. 顶部工具栏

顶部工具栏从左到右包含以下按钮：

| 按钮     | 功能                         | 状态规则                     |
| -------- | ---------------------------- | ---------------------------- |
| 选取     | 切换到画布平移工具           | 激活时显示主色背景           |
| 房间     | 切换到房间绘制工具           | 默认激活，激活时显示主色背景 |
| 添加房间 | 在当前视口中心添加默认房间   | 始终可用                     |
| 撤销     | 撤销最近一次内容变更         | 没有可撤销记录时禁用         |
| 重做     | 恢复最近一次被撤销的内容变更 | 没有可重做记录时禁用         |
| 清空画板 | 删除全部房间                 | 画布为空时禁用               |

工具栏本身会阻止指针事件冒泡，因此点击或拖动按钮不会在画布上创建房间。

## 4. 创建房间

### 4.1 拖拽创建

1. 选择顶部“房间”工具。
2. 在画布空白处按下鼠标左键或主指针。
3. 拖拽形成矩形预览。
4. 松开指针完成创建。

创建过程默认按 20 像素网格吸附。矩形宽度或高度小于 20 像素时不会创建房间。

### 4.2 快速添加

点击顶部“添加房间”按钮，会在当前视口中心创建一个 6 × 4 格的默认房间。画布已经平移或缩放时，系统仍会按当前视口中心计算房间位置。

### 4.3 默认名称

房间按创建顺序命名为“房间 1”“房间 2”等。删除或清空房间不会回退已经使用的序号，刷新页面后序号也会继续递增。

## 5. 选择与编辑房间

### 5.1 选择

- 点击房间或通过键盘聚焦房间即可选中。
- 选中房间显示朱砂红边框、轻量阴影和八个缩放手柄。
- 点击画布并开始绘制新房间时会取消原房间选择。
- 按 `Escape` 可取消当前选择。

### 5.2 移动

- 使用指针拖拽房间可调整位置。
- 使用方向键可按一个网格移动房间。
- 移动过程默认吸附到网格。

### 5.3 调整大小

- 拖拽选中房间周围的八个手柄可调整对应方向的边界。
- 按 `Shift + 方向键` 可按一个网格调整宽度或高度。
- 房间最小宽度和高度均为 20 像素。

## 6. 创建和编辑家具

### 6.1 创建家具

1. 选择目标房间，右侧自动打开当前位置详情。
2. 点击“添加家具”，详情面板关闭并进入一次性家具绘制模式。
3. 在目标房间内部拖拽形成家具矩形，松开后完成创建。
4. 创建成功后自动退出绘制模式并选中新家具；按 `Escape` 可提前取消。

家具默认命名为“家具 1”“家具 2”等，坐标相对于所属房间保存。家具不能在房间外创建，尺寸不足 20 像素时不会创建。

### 6.2 移动和调整大小

- 指针拖拽或方向键移动家具。
- 八方向手柄或 `Shift + 方向键` 调整家具大小。
- 家具位置和尺寸始终限制在所属房间内。
- 房间不能缩小到无法容纳已有家具或房间级储物设备；移动房间时子实体自然跟随。

### 6.3 家具详情

选中家具后，右侧详情显示完整路径、家具名称、网格尺寸和所属房间。用户可以在详情中重命名、删除家具，或通过“所属房间”选择框把家具转移到其他房间。转移后家具会在新房间中居中并保持边界合法。

### 6.4 房间详情和级联删除

房间详情显示当前家具和房间级储物设备数量及列表，可以直接定位具体实体。删除包含子内容的房间时会先显示受影响的家具和储物设备数量；确认后作为一次操作级联删除，可通过一次撤销完整恢复。

## 7. 创建和管理储物设备

### 7.1 房间级储物设备

1. 选择目标房间，在当前位置详情中点击“添加储物设备”。
2. 详情面板关闭并进入一次性绘制模式。
3. 在目标房间内拖拽形成矩形，松开后创建并选中新设备。
4. 按 `Escape` 可取消绘制。

房间级储物设备默认命名为“储物设备 1”“储物设备 2”等。它支持指针或方向键移动、八方向手柄或 `Shift + 方向键` 调整大小，位置和尺寸始终限制在所属房间内。

### 7.2 家具内储物设备

选择家具后点击“添加储物设备”，系统会立即创建一个家具内逻辑储物设备。它显示在家具详情列表中，不在画布上绘制矩形，也不保存无意义的家具内部坐标。

### 7.3 位置切换

储物设备详情中的“存放位置”选择器按房间分组，允许选择：

- 直接放在某个房间。
- 放入该房间中的某件家具。

房间级设备移入家具时，画布矩形立即消失；家具内设备移入房间时，会在目标房间中心生成默认矩形并保证不越界。移到另一房间或家具后，详情路径和搜索路径立即更新。

### 7.4 详情与级联

储物设备详情显示完整路径、名称、位置类型和可用操作。设备可重命名、删除和转移位置。删除房间或家具时，其内部储物设备会计入影响数量并一起删除，整次级联可一次撤销。

## 8. 添加和管理物品

### 8.1 快速添加

房间、家具和储物设备详情都提供“添加物品”按钮。打开表单后，当前实体自动作为默认存放位置，用户填写物品名称和正整数数量即可保存，也可以直接改选其他房间、家具或储物设备。

### 8.2 编辑、移动和删除

各实体详情会列出直接存放在当前位置的物品及数量。点击“编辑”可以同时修改名称、数量和存放位置；位置改变后，物品会立即出现在新位置的详情列表中。点击“删除”会移除该物品，以上操作均可撤销和重做。

### 8.3 搜索定位

物品与其他实体共用右上角搜索。结果显示“房间 / 家具 / 储物设备 / 物品”形式的实时路径；点击结果后会定位到最终容器并在详情列表中短暂高亮物品。

## 9. 重命名、删除与清空

### 9.1 重命名

可通过以下方式打开重命名弹窗：

- 在房间上右击并选择“修改名称”。
- 房间获得焦点后按 `Enter`。
- 房间获得焦点后按 `F2`。

弹窗会显示房间当前名称。确认后更新名称，取消或按 `Escape` 不修改名称。

家具和房间级储物设备使用相同的右键菜单、`Enter` / `F2` 和名称输入流程；家具内储物设备从详情面板重命名。

### 9.2 删除实体

可通过以下方式删除房间：

- 在房间上右击并选择“删除房间”。
- 房间获得焦点后按 `Delete` 或 `Backspace`。
- 选中房间后直接按 `Delete` 或 `Backspace`。
- 家具可通过右键菜单、详情面板或键盘删除。
- 房间级储物设备可通过右键菜单、详情面板或键盘删除，家具内储物设备从详情面板删除。

### 9.3 清空画板

可通过以下方式删除全部房间：

- 点击顶部“清空画板”按钮。
- 在画布空白处右击并选择“清空画板”。

当前清空操作直接执行，不显示二次确认弹窗。清空操作可以通过撤销恢复。

## 10. 画布平移与缩放

### 10.1 平移画布

1. 点击顶部“选取”按钮。
2. 在画布空白区域按下并拖拽。
3. 松开指针结束平移。

画布平移只改变视口，不改变房间坐标，也不会进入撤销历史。

### 10.2 缩放画布

- 在画布上滚动滚轮可缩放。
- 点击右下角减号按钮可缩小。
- 点击右下角加号按钮可放大。
- 当前缩放比例显示在两个按钮中间。

缩放范围为 50% 至 200%，步长为 10%。缩放以画布视口中心为锚点，不进入撤销历史。

## 11. 撤销与重做

系统最多保存 50 条完整画板内容历史；默认命名计数器不随撤销回退。

会记录的操作：

- 创建房间。
- 移动房间。
- 调整房间大小。
- 修改房间名称。
- 删除房间。
- 清空画板。
- 创建、移动、缩放、重命名、转移或删除家具。
- 创建、移动、缩放、重命名、转移或删除储物设备。
- 添加、修改、移动或删除物品。
- 包含家具、储物设备或物品的父实体级联删除。

不会记录的操作：

- 选择或取消选择房间。
- 切换选取/房间工具。
- 平移或缩放画布。
- 搜索、定位和高亮实体。
- 打开或关闭菜单、面板和弹窗。

连续拖拽移动或缩放一次只产生一条历史记录。执行新的房间编辑后，原有重做记录会被清空。

## 12. 搜索、全部房间与当前位置

### 12.1 搜索全部实体

1. 点击右上角“搜索”按钮。
2. 输入房间、家具、储物设备或物品名称的全部或部分内容。
3. 点击搜索结果定位目标。

搜索忽略大小写，空搜索词会显示当前四类实体。房间结果展示网格尺寸，家具、储物设备和物品结果展示当前所属路径。

搜索面板支持以下键盘操作：

- `ArrowDown` / `ArrowUp`：在结果之间循环移动焦点。
- `Enter`：输入框聚焦时定位第一个结果。
- `Escape`：关闭面板并把焦点还给搜索按钮。

### 12.2 全部房间

点击右上角房间图标可查看全部房间。入口的无障碍名称会同时显示当前房间数量。点击列表中的房间即可定位。

### 12.3 当前位置详情

选中房间、家具或储物设备会打开当前位置详情。右上角第三个按钮可以重新打开已关闭的详情面板；没有选择时该按钮禁用。

### 12.4 定位效果

定位房间时会执行以下操作：

1. 选中目标实体。
2. 将目标所属房间移动到画布视口中心。
3. 在画布节点、逻辑储物设备详情或物品列表中短暂显示高亮提示。
4. 优先保持当前缩放比例；目标过大时自动缩小到可完整展示的比例。
5. 关闭搜索或全部房间面板。

用户在定位动画期间开始绘制、移动、调整大小或平移画布时，定位动画会立即停止。

## 13. 右键菜单与浮层关闭规则

### 13.1 画布菜单

在画布空白处右击，显示“添加房间”和“清空画板”。没有房间时清空项禁用。

### 13.2 空间实体菜单

在房间上右击，显示：

- 添加家具。
- 添加储物设备。
- 添加物品。
- 修改名称或删除房间。

家具菜单提供添加储物设备、添加物品、修改名称和删除家具；房间级储物设备菜单提供添加物品、修改名称和删除储物设备。

### 13.3 关闭规则

- 点击菜单之外的任意区域关闭右键菜单。
- 按 `Escape` 关闭右键菜单。
- 点击搜索/房间面板之外的区域关闭对应面板。
- 通过 `Escape` 关闭菜单或面板后，焦点会恢复到相应触发元素；点击外部区域时直接关闭浮层。

## 14. 快捷键汇总

| 快捷键                            | 功能                     |
| --------------------------------- | ------------------------ |
| `ArrowLeft/Right/Up/Down`         | 移动获得焦点的空间实体   |
| `Shift + ArrowLeft/Right/Up/Down` | 调整获得焦点实体的尺寸   |
| `Enter` / `F2`                    | 重命名获得焦点的空间实体 |
| `Delete` / `Backspace`            | 删除获得焦点的空间实体   |
| `ContextMenu` / `Shift + F10`     | 打开当前实体右键菜单     |
| `Escape`                          | 取消选择或关闭当前浮层   |
| `Ctrl/Cmd + Z`                    | 撤销                     |
| `Ctrl/Cmd + Shift + Z`            | 重做                     |
| `Ctrl + Y`                        | 重做                     |

输入框中输入文字时，全局删除、撤销和重做快捷键不会误操作画板。

## 15. 本地保存

- 页面使用 `CanvasDocumentV2` 把房间、家具、储物设备、物品和各实体序号保存在当前浏览器的 `localStorage`。
- 画板内容变更后约 300 毫秒写入一次，离开页面时会再次保存最新数据。
- 刷新页面或重新打开同一站点时自动恢复当前实体和层级关系。
- 旧版房间数据读取后在内存迁移，后续保存统一写入 version 2。
- 存储内容损坏或版本不兼容时，系统忽略该数据并显示空画板。
- 数据只存在当前浏览器和当前站点环境中，清理浏览器站点数据后无法恢复。
- 选中状态、画布位置、缩放比例和撤销/重做记录不会跨刷新保留。

## 16. 响应式与无障碍

- 小屏设备上，右上角功能区会调整位置和宽度，避免与顶部工具栏重叠。
- 粗指针设备上的主要按钮使用至少 44 × 44 像素的点击区域。
- 房间、家具、储物设备、详情控件、工具按钮、搜索输入框、菜单和弹窗均可通过键盘访问。
- 页面提供屏幕阅读器操作说明和动态结果数量提示。
- 用户启用“减少动态效果”后，定位和高亮动画会被关闭或弱化。

## 17. 当前限制

- 物品只在详情列表中展示，不作为画布矩形节点渲染。
- 家具内储物设备只维护逻辑层级，不支持家具内部二维布局。
- 不支持账号、云同步和多人协作。
- 不支持导入、导出、分享或打印布局。
- 不支持复制、旋转、组合、对齐和图层排序。
- 清空画板和删除空实体直接执行；删除包含子内容的房间或家具会显示影响数量并要求确认。

## 18. 文档同步要求

- 用户可见功能新增、删除或行为调整时，必须同步更新本文档和《需求文档》。
- 快捷键、按钮顺序、禁用状态、默认值、范围或交互细节变化时，必须在对应章节同步修改。
- 数据保存方式或刷新恢复行为变化时，必须同时更新本文档和《架构设计文档》。
- 文档内容必须与已经合入的代码一致，不记录尚未实现的功能。
