# 家居布局画板功能文档

## 1. 文档说明

- 文档状态：当前功能说明
- 适用版本：`zhijian-hub@0.1.0`
- 最后核对：2026-07-18
- 维护范围：当前页面功能、操作方式和用户可见行为
- 关联代码：`src/components/features/room-canvas/`、`src/components/ui/`

本文档面向产品、测试和开发人员，说明当前页面已经提供的功能、操作方式和行为规则。

## 2. 页面组成

当前项目只有一个家居布局画板页面，主要由以下区域组成：

| 区域       | 位置         | 作用                                    |
| ---------- | ------------ | --------------------------------------- |
| 品牌标识   | 左上角       | 展示“知简”品牌                          |
| 顶部工具栏 | 顶部居中     | 切换工具、添加房间、撤销/重做、清空画板 |
| 画板功能区 | 右上角       | 搜索房间、查看全部房间                  |
| 画布       | 页面主体     | 绘制、选择、移动和调整房间              |
| 缩放控件   | 右下角       | 缩小、显示比例、放大                    |
| 右键菜单   | 指针位置附近 | 执行画板或房间操作                      |
| 重命名弹窗 | 页面中央     | 修改房间名称                            |

## 3. 顶部工具栏

顶部工具栏从左到右包含以下按钮：

| 按钮     | 功能                         | 状态规则                     |
| -------- | ---------------------------- | ---------------------------- |
| 选取     | 切换到画布平移工具           | 激活时显示主色背景           |
| 房间     | 切换到房间绘制工具           | 默认激活，激活时显示主色背景 |
| 添加房间 | 在当前视口中心添加默认房间   | 始终可用                     |
| 撤销     | 撤销最近一次房间变更         | 没有可撤销记录时禁用         |
| 重做     | 恢复最近一次被撤销的房间变更 | 没有可重做记录时禁用         |
| 清空画板 | 删除全部房间                 | 画布为空时禁用               |

工具栏本身会阻止指针事件冒泡，因此点击或拖动按钮不会在画布上创建房间。

## 4. 创建房间

### 4.1 拖拽创建

1. 选择顶部“房间”工具。
2. 在画布空白处按下鼠标左键或主指针。
3. 拖拽形成矩形预览。
4. 松开指针完成创建。

创建过程默认按 20 像素网格吸附。矩形宽度或高度小于 20 像素时不会创建房间。

### 4.2 快速添加

点击顶部“添加房间”按钮，会在当前视口中心创建一个 6 × 4 格的默认房间。画布已经平移或缩放时，系统仍会按当前视口中心计算房间位置。

### 4.3 默认名称

房间按创建顺序命名为“房间 1”“房间 2”等。删除或清空房间不会回退已经使用的序号，刷新页面后序号也会继续递增。

## 5. 选择与编辑房间

### 5.1 选择

- 点击房间或通过键盘聚焦房间即可选中。
- 选中房间显示朱砂红边框、轻量阴影和八个缩放手柄。
- 点击画布并开始绘制新房间时会取消原房间选择。
- 按 `Escape` 可取消当前选择。

### 5.2 移动

- 使用指针拖拽房间可调整位置。
- 使用方向键可按一个网格移动房间。
- 移动过程默认吸附到网格。

### 5.3 调整大小

- 拖拽选中房间周围的八个手柄可调整对应方向的边界。
- 按 `Shift + 方向键` 可按一个网格调整宽度或高度。
- 房间最小宽度和高度均为 20 像素。

## 6. 重命名、删除与清空

### 6.1 重命名

可通过以下方式打开重命名弹窗：

- 在房间上右击并选择“修改名称”。
- 房间获得焦点后按 `Enter`。
- 房间获得焦点后按 `F2`。

弹窗会显示房间当前名称。确认后更新名称，取消或按 `Escape` 不修改名称。

### 6.2 删除房间

可通过以下方式删除房间：

- 在房间上右击并选择“删除房间”。
- 房间获得焦点后按 `Delete` 或 `Backspace`。
- 选中房间后直接按 `Delete` 或 `Backspace`。

### 6.3 清空画板

可通过以下方式删除全部房间：

- 点击顶部“清空画板”按钮。
- 在画布空白处右击并选择“清空画板”。

当前清空操作直接执行，不显示二次确认弹窗。清空操作可以通过撤销恢复。

## 7. 画布平移与缩放

### 7.1 平移画布

1. 点击顶部“选取”按钮。
2. 在画布空白区域按下并拖拽。
3. 松开指针结束平移。

画布平移只改变视口，不改变房间坐标，也不会进入撤销历史。

### 7.2 缩放画布

- 在画布上滚动滚轮可缩放。
- 点击右下角减号按钮可缩小。
- 点击右下角加号按钮可放大。
- 当前缩放比例显示在两个按钮中间。

缩放范围为 50% 至 200%，步长为 10%。缩放以画布视口中心为锚点，不进入撤销历史。

## 8. 撤销与重做

系统最多保存 50 条房间状态历史。

会记录的操作：

- 创建房间。
- 移动房间。
- 调整房间大小。
- 修改房间名称。
- 删除房间。
- 清空画板。

不会记录的操作：

- 选择或取消选择房间。
- 切换选取/房间工具。
- 平移或缩放画布。
- 搜索、定位和高亮房间。
- 打开或关闭菜单、面板和弹窗。

连续拖拽移动或缩放一次只产生一条历史记录。执行新的房间编辑后，原有重做记录会被清空。

## 9. 搜索与全部房间

### 9.1 搜索房间

1. 点击右上角“搜索”按钮。
2. 输入房间名称的全部或部分内容。
3. 点击搜索结果定位房间。

搜索忽略大小写，空搜索词会显示全部房间。结果中展示房间名称和房间的网格尺寸。

搜索面板支持以下键盘操作：

- `ArrowDown` / `ArrowUp`：在结果之间循环移动焦点。
- `Enter`：输入框聚焦时定位第一个结果。
- `Escape`：关闭面板并把焦点还给搜索按钮。

### 9.2 全部房间

点击右上角房间图标可查看全部房间。入口的无障碍名称会同时显示当前房间数量。点击列表中的房间即可定位。

### 9.3 定位效果

定位房间时会执行以下操作：

1. 选中目标房间。
2. 将房间中心移动到画布视口中心。
3. 短暂显示高亮光晕。
4. 优先保持当前缩放比例；目标过大时自动缩小到可完整展示的比例。
5. 关闭搜索或全部房间面板。

用户在定位动画期间开始绘制、移动、调整大小或平移画布时，定位动画会立即停止。

## 10. 右键菜单与浮层关闭规则

### 10.1 画布菜单

在画布空白处右击，显示“清空画板”。没有房间时该菜单项禁用。

### 10.2 房间菜单

在房间上右击，显示：

- 修改名称。
- 删除房间。

### 10.3 关闭规则

- 点击菜单之外的任意区域关闭右键菜单。
- 按 `Escape` 关闭右键菜单。
- 点击搜索/房间面板之外的区域关闭对应面板。
- 通过 `Escape` 关闭菜单或面板后，焦点会恢复到相应触发元素；点击外部区域时直接关闭浮层。

## 11. 快捷键汇总

| 快捷键                            | 功能                   |
| --------------------------------- | ---------------------- |
| `ArrowLeft/Right/Up/Down`         | 移动获得焦点的房间     |
| `Shift + ArrowLeft/Right/Up/Down` | 调整获得焦点房间的尺寸 |
| `Enter` / `F2`                    | 重命名房间             |
| `Delete` / `Backspace`            | 删除房间               |
| `ContextMenu` / `Shift + F10`     | 打开房间右键菜单       |
| `Escape`                          | 取消选择或关闭当前浮层 |
| `Ctrl/Cmd + Z`                    | 撤销                   |
| `Ctrl/Cmd + Shift + Z`            | 重做                   |
| `Ctrl + Y`                        | 重做                   |

输入框中输入文字时，全局删除、撤销和重做快捷键不会误操作画板。

## 12. 本地保存

- 页面会把房间列表和下一房间序号保存在当前浏览器的 `localStorage`。
- 房间变更后约 300 毫秒写入一次，离开页面时会再次保存最新数据。
- 刷新页面或重新打开同一站点时自动恢复房间。
- 存储内容损坏或版本不兼容时，系统忽略该数据并显示空画板。
- 数据只存在当前浏览器和当前站点环境中，清理浏览器站点数据后无法恢复。
- 选中状态、画布位置、缩放比例和撤销/重做记录不会跨刷新保留。

## 13. 响应式与无障碍

- 小屏设备上，右上角功能区会调整位置和宽度，避免与顶部工具栏重叠。
- 粗指针设备上的主要按钮使用至少 44 × 44 像素的点击区域。
- 房间、工具按钮、搜索输入框、菜单和弹窗均可通过键盘访问。
- 页面提供屏幕阅读器操作说明和动态结果数量提示。
- 用户启用“减少动态效果”后，定位和高亮动画会被关闭或弱化。

## 14. 当前限制

- 画板目前只支持房间矩形，不支持家具或其他实体。
- 不支持账号、云同步和多人协作。
- 不支持导入、导出、分享或打印布局。
- 不支持复制、旋转、组合、对齐和图层排序。
- 清空画板和删除房间没有二次确认，主要依赖撤销功能恢复。

- 用户可见功能新增、删除或行为调整时，必须同步更新本文档和《需求文档》。
