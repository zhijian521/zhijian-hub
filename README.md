# 知简 Hub

基于 Next.js 15 的家居布局画板，提供房间规划、搜索定位、撤销重做和浏览器本地保存能力。

## ✨ 核心特性

- **房间布局画板** - 支持创建、移动、缩放、重命名、删除和清空房间
- **画布交互** - 支持平移、缩放、网格吸附、搜索定位和键盘操作
- **编辑保障** - 支持撤销/重做和版本化 localStorage 持久化
- **Next.js 15 + App Router** - React 单页面应用架构
- **TypeScript 5** - 严格类型检查
- **中国风简约主题** - 文人书斋·朱砂红·宣纸白
- **完整规范体系** - 涵盖开发、样式、API、Git、测试等规范
- **可执行规范** - 每项规范都有对应配置和脚本支持

## 🎨 主题系统

内置中式简约亮色主题，包含完整的设计令牌体系：

- **主色调**：朱砂红 `#9f000f` - 文人书斋的典雅
- **副色调**：竹青 `#4a5d23` - 自然清新气息
- **背景色**：宣纸白 `#fbf9f9` - 素净温和
- **文字色**：墨色 `#1d1b20` - 经典文字系统
- **字体族**：衬线体标题 + 无衬线正文

## 🏗 项目结构

```
src/
├── app/                   # Next.js App Router
├── styles/               # 全局样式系统
│   ├── tokens.css        # 设计令牌
│   └── theme.css         # 主题变量
├── components/           # UI、站点与业务组件
└── lib/                  # Hooks、类型与工具函数

```
