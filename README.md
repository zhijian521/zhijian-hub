# 知简 Hub

基于 Next.js 15 的通用项目起始模板，内置完整开发规范和中式简约主题。

## ✨ 核心特性

- **Next.js 15 + App Router** - 现代 React 框架架构
- **TypeScript 5** - 严格类型检查，提升代码质量
- **中国风简约主题** - 文人书斋·朱砂红·宣纸白
- **完整规范体系** - 涵盖开发、样式、API、Git、测试等7大规范
- **可执行规范** - 每项规范都有对应配置和脚本支持

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 生产构建

```bash
npm run build
npm run start
```

## 🛠 开发工具

### 代码质量检查

```bash
# 执行 lint + typecheck
npm run check
```

### 代码格式化

```bash
# 格式化所有文件
npm run format

# 检查格式
npm run format:check
```

## 📋 开发规范

项目内置 7 套完整规范，确保代码质量和一致性：

| 编号 | 规范       | 说明                   |
| ---- | ---------- | ---------------------- |
| 00   | [项目总则] | 目标、技术栈、边界定义 |
| 01   | [目录结构] | Next.js 项目结构标准   |
| 02   | [代码风格] | TypeScript 编码规范    |
| 03   | [样式规范] | CSS 组织和样式最佳实践 |
| 04   | [API规范]  | 接口设计和数据标准     |
| 05   | [组件规范] | 组件设计和实现标准     |
| 06   | [Git规范]  | 分支和提交规范         |
| 07   | [测试规范] | 测试策略和执行标准     |

详细规范参见 [specs/README.md](./specs/README.md)。

## 🎨 主题系统

内置中式简约亮色主题，包含完整的设计令牌体系：

- **主色调**：朱砂红 `#9f000f` - 文人书斋的典雅
- **副色调**：竹青 `#4a5d23` - 自然清新气息
- **背景色**：宣纸白 `#fbf9f9` - 素净温和
- **文字色**：墨色 `#1d1b20` - 经典文字系统
- **字体族**：衬线体标题 + 无衬线正文

## 🏗 项目结构

```
src/
├── app/                   # Next.js App Router
├── styles/               # 全局样式系统
│   ├── tokens.css        # 设计令牌
│   └── theme.css         # 主题变量
├── components/           # React 组件库（后续扩展）
└── lib/                  # 工具库（后续扩展）
```

## 🧪 提交规范

所有提交前必须通过代码质量门禁：

```bash
npm run check
```

确保：

- ✅ ESLint 检查通过
- ✅ TypeScript 类型检查通过
- ✅ 代码格式化符合规范

## 📝 技术栈

- **框架**：Next.js 15.3.8 (App Router)
- **语言**：TypeScript 5.8.3
- **样式**：CSS Modules + CSS 变量
- **工具链**：ESLint + Prettier
- **辅助依赖**：clsx (类名组合工具)
